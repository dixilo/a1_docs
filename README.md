# 課題演習A1: コンプトン散乱

PMT・シンチレータを用いた予備実験から、CAMAC ADC によるスペクトル測定、
コンプトン散乱の本測定までを説明する学生向け資料です。

内容としては、シンチレータと光電子増倍管の基礎、NIM/CAMAC を使った信号処理、
スペクトルの見方、較正、コンプトン散乱測定のセットアップと手順、
インピーダンスやCAMAC規格に関する付録を含みます。

本文は `main.tex`、画像や図版は `figures/` にあります。

## ビルド

```sh
latexmk main.tex
```

生成物は `main.pdf` です。

クリーンアップする場合は以下を使います。

```sh
latexmk -C
```
